# Base box Requirements

## Operating System

* SamuraiWTF runs on Linux. A base box for the WTF framework should run on:
  * Amazon Linux, specifically the latest AWS Workspaces version
  * Debian / Ubuntu (?TBD), when using Vagrant or PackerIO


## Installed Packages

* nginx
* vim
* curl
* docker
* unzip
* python-pip
* php-fpm
* nano
